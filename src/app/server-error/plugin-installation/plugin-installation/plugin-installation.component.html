Steps to install the Share screen and Record screen plugin on your PC.
  1. Go to Login page of Registered User/ Guest User.
  2. If you have not added this site to your exception list of your chrome browser, then it will show error message as “Pop-up Blocker is enabled! Please add this site to your exception list, and refresh the page.”
  3. Click on pop up blocked icon , select Always allow pop-ups and redirects from https://cfscommunicator.com and click on Done button as shown in the below image.
  4. Refresh the page.
  5. A pop–up as shown in below screen will appear to install the screen share chrome plugin. Click on the link provided in the pop up.
  6. Click on Add to Chrome button as shown in below screen.
  7. Click on Add extension button as shown in below screen.
  8. After adding the Screen Capturing plugin successfully, it will show a message as “ Screen Capturing has been added to Chrome ” and screen capturing plugin icon will get added to your chrome browser as shown in the below screen.
  9. Close the Screen Capturing – Chrome Web Store – Google Chrome pop up.
  10. After closing the pop up, an another pop up to install the record screen chrome plugin as shown in below screen will appear. Click on the link provided in the pop up.
  11. Click on Add to Chrome button as shown in below screen.
  12. Click on Add extension button as shown in below screen.
  13. After adding the RecordRTC plugin successfully, it will show a message as “ RecordRTC has been added to Chrome ” and a RecordRTC plugin icon will get added to the browser as shown in the below screen.
  14. Close the RecordRTC – Chrome Web Store – Google Chrome pop up. Now you can login with your credentials and proceed further.
