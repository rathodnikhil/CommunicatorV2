Steps to save recorded videos on your browser. Attach screenshots and do makeup of this page!!
  1. Click on Record RTC icon as shown in the below screen.
  2. Click on See All Recordings, it will open the page as shown in the below screen.Here you can see your latest recored video(Meeting recording) files.
  3. Click on Menu button,to see the list of recordings(your recorded files).The list as shown in the below image, shows all the recorded files along with the date of recording.
  4. To Edit the name of the recording,click on edit icon.On click of edit button,it will show a pop up as shown in the below screen.In the pop up, edit the name of the file and click on ok button,your file name gets edited.
  5. To Delete the recording,click on delete icon.On click of delete icon,it will show a pop up as shown in the below screen.Click on ok button of the pop up to delete the recording permanently.
  6. To download the recording,click on More icon of the recording video as shown in the below screen OR click on link here of "click here to download".
  7. You can upload the recording by clicking on upload icon.On click of upload button,it will provide two options as shown in below image.
    1. Upload to Private Server: You can upload the recored file on private server
    2. Upload to YouTube:You can upload the recored file on YouTube.
